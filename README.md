# MoneyForward Asset Graph

マネーフォワード ME の資産推移を美しいグラフで可視化する Google Chrome 拡張機能です。過去データのCSV一括ダウンロードも可能。

## ✨ 主な機能

### 1. 資産推移グラフ

- **資産合計推移**: 資産総額の変動を折れ線グラフで直感的に把握できます
- **内訳の積み上げ表示**: 「預金・現金」「株式」「投資信託」などの内訳を積み上げグラフに切り替え可能
- **柔軟なフィルタリング**:
  - 期間指定: 過去1年 / 3年 / 5年 / 10年 / 全期間
  - 特定日抽出: 「毎月〇日」の条件でデータを絞り込み

### 2. CSV 一括ダウンロード

- **全日次データ**: 過去の全データを1つのCSVファイルとして保存
- **指定日抽出**: 「毎月25日（給料日）」や「毎月末」など、特定の日付のデータだけを抽出
- **高速処理**: 並列リクエストにより、10年分のデータも数十秒で取得可能
- **キャッシュ機能**: 過去月のデータはブラウザに保存され、次回以降は高速で読み込み

### 3. データのエクスポート

- **CSV保存**: グラフデータをCSVとしてダウンロード
- **画像コピー**: グラフを画像としてクリップボードにコピー
- **データコピー**: Excel貼り付け用にタブ区切りテキストとしてコピー

### 4. テーマカスタマイズ

20種類のプリセットから好みのカラーテーマを選択可能

## 📦 インストール方法

### Chrome ウェブストアから（推奨）

[Chrome ウェブストアのリンク]（審査通過後に追加予定）

### 開発者向け（手動インストール）

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 🚀 使い方

1. マネーフォワード ME の[資産推移画面](https://moneyforward.com/bs/history)にアクセス
2. 画面右下に操作パネルが表示されます
3. **CSVダウンロード**: 期間を選んでCSVボタンをクリック
4. **グラフ分析**: 「グラフを表示・生成」をクリックしてモーダル画面で分析

## 🔒 プライバシー

この拡張機能は：

- ✅ すべての処理をブラウザ内で完結
- ✅ 外部サーバーへのデータ送信なし
- ✅ ユーザーデータの収集なし

詳細は [プライバシーポリシー](PRIVACY_POLICY.md) をご覧ください。

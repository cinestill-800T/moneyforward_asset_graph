# MoneyForward Asset Downloader

マネーフォワード ME の資産推移データを一括でCSVダウンロードするための Google Chrome 拡張機能です。
通常の機能では取得できない「過去の全日次データ」や「毎月特定日（25日など）のデータ」を高速に一括取得できます。

## 特徴

*   **🚀 高速一括ダウンロード**: 過去数年分の資産データを並列処理で高速に取得します。
*   **📊 全日次データ取得**: 月末だけでなく、CSVに含まれる全日次の資産データを結合して出力します。
*   **📅 特定日抽出**: 「毎月25日」など、指定した日付のデータのみを抽出してCSV化する機能も搭載。
*   **🎨 モダンなUI**: 直感的に操作できる美しいグラデーションUI。

## インストール方法

本拡張機能はChromeウェブストアには公開されていません。以下の手順で手動インストールしてください。

1.  このリポジトリをクローンまたはダウンロード（ZIP解凍）します。
2.  Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力します。
3.  右上の「デベロッパーモード」スイッチをオンにします。
4.  左上の「パッケージ化されていない拡張機能を読み込む」をクリックします。
5.  ダウンロードしたフォルダ（`manifest.json` が含まれているフォルダ）を選択します。

## 使い方

1.  マネーフォワード ME の [資産推移画面](https://moneyforward.com/bs/history) にアクセスします。
2.  画面右下に操作パネルが表示されます。
3.  **取得範囲**を選択します（過去1年〜全期間）。
4.  目的に合わせてボタンをクリックします。
    *   **全日次データを一括取得**: 全期間のすべての日のデータを1つのCSVにします。
    *   **指定日のみ抽出して保存**: 「毎月〇日」のデータだけを抜き出してCSVにします。

## ファイル構成

```
.
├── assets/          # アイコン画像
├── content.js       # メインロジック（データ取得・UI制御）
├── style.css        # UIスタイル定義
├── manifest.json    # 拡張機能設定ファイル
└── README.md        # 本ドキュメント
```

## 注意事項

*   本拡張機能は非公式ツールです。マネーフォワード社の公式ツールではありません。
*   大量のデータ取得を行うため、短時間に過度な実行を繰り返すとサーバーに負荷がかかる可能性があります。常識の範囲内でご利用ください。
*   取得したデータはShift_JIS形式のCSVとして保存されます。Excel等でそのまま開けます。

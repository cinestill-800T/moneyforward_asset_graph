# MoneyForward Asset Graph

マネーフォワード ME の資産推移を美しいグラフで可視化する Google Chrome 拡張機能です。過去データのCSV一括ダウンロードも可能。

## ✨ 主な機能

### 📊 資産推移グラフ（メイン機能）

資産総額の変動を美しいグラフで可視化します。

- **折れ線グラフ**: 資産合計の推移を直感的に把握
- **積み上げグラフ**: 預金・株式・投資信託などの内訳表示
- **増減（差分）グラフ**: 前回からの資産増減を棒グラフで表示（NEW!）
- **柔軟なフィルタリング**: 期間指定、特定日抽出、月末判定など
- **20種類のカラーテーマ**: お好みの配色でグラフを表示

### 📥 CSVエクスポート

グラフで確認したデータをそのままCSVで保存できます。

- グラフ画面から直接ダウンロード
- 画像コピー・データコピーにも対応
- 高速キャッシュにより10年分でも数十秒で取得

## 📦 インストール方法

### Chrome ウェブストアから（推奨）

[Chrome ウェブストアのリンク]（審査通過後に追加予定）

### 開発者向け（手動インストール）

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 🚀 使い方

1. マネーフォワード ME の[資産推移画面](https://moneyforward.com/bs/history)にアクセス
2. 画面右下に表示される「**グラフを表示**」ボタンをクリック
3. グラフ画面で期間やフィルターを設定し、「再取得・描画」で表示
4. 必要に応じてCSVダウンロードや画像コピーを実行

## 🔒 プライバシー

- ✅ すべての処理をブラウザ内で完結
